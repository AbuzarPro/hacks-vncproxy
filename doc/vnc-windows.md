# Connexion reverse VNC sous windows

## TightVNC

Pour le serveur VNC sous Windows j'ai choisi TightVNC car il est libre et dispose d'une bonne communauté en cas de besoin.

## Configuration

Une fois installé et lancé, une petite icône apparaît en bas à droite dans la barre de tâches. Il suffit de faire un clic droit dessus et de cliquer sur `attach listening viewer`.

Là une boite de dialogue apparaît dans laquelle il faut rentrer l'adresse de la machine avec laquelle on veut partager la connexion ainsi que le port sur lequel le serveur va écouter par exemple : `192.168.1.42::443`
